🚨 ВИКОНАЙТЕ ЦІ КОМАНДИ ЧЕРЕЗ SSH ЯК ROOT
==========================================

КОМАНДА 1: Перейти в папку проекту
-----------------------------------
cd /var/www/pashland/data/www/g-shop.info


КОМАНДА 2: Оновити код
----------------------
git pull origin main


КОМАНДА 3: Перейти в backend
-----------------------------
cd backend


КОМАНДА 4: Видалити старі модулі та встановити нові
----------------------------------------------------
rm -rf node_modules
npm install
npm install @strapi/strapi@4.25.13


КОМАНДА 5: Створити .env файл
------------------------------
cat > .env << 'EOF'
NODE_ENV=production
HOST=0.0.0.0
PORT=3002
APP_KEYS=toBeModified1,toBeModified2
API_TOKEN_SALT=tobemodified
ADMIN_JWT_SECRET=tobemodified
JWT_SECRET=QWJjMTIzNDU2Nzg5MDEyMzQ1Njc4OTAxMjM0NTY3ODkwMTI=
DATABASE_CLIENT=sqlite
DATABASE_FILENAME=./database.db
EOF


КОМАНДА 6: Побудувати Strapi
-----------------------------
npm run build


КОМАНДА 7: Запустити Strapi напряму
------------------------------------
PORT=3002 npm run start &


АБО АЛЬТЕРНАТИВНО (якщо попереднє не працює):
---------------------------------------------
NODE_ENV=production HOST=0.0.0.0 PORT=3002 npx strapi start &


КОМАНДА 8: Перевірити чи працює
--------------------------------
curl http://localhost:3002/api/products
